# Product Decisions

Small, durable decision records complement versioned PRDs.

- Naming: `PD-001.md`, `PD-002.md`, …
- Store product decisions that should survive beyond chat.
- Reference decisions from PRD sections instead of losing rationale in notes only.
- Product decisions are project-scoped generated docs.
- The reusable template lives in `.cursor/core/templates/prd/product-decision.template.md`.
