# Playbook express — « Pitch demain »

## Quand l’utiliser

Deadline **&lt; 24 h** : meeting investisseur, associé, advisor, accélérateur — besoin d’un **livrable partageable**, pas d’un PRD académique.

---

## Checklist 30 min (fondateur)

| Min | Action | Done |
|-----|--------|------|
| 0–2 | Créer projet → **I need it today** (express) | ☐ |
| 2–15 | Clarify : répondre aux mini-forms (viser ≤ 6 tours) | ☐ |
| 15–22 | **Generate express livrable** (12 sections lean) | ☐ |
| 22–25 | Relire `executive_summary` + `differentiation` | ☐ |
| 25–28 | **Share link** + tester en navigation privée (disclaimer visible) | ☐ |
| 28–30 | Copier lien dans deck / email | ☐ |

---

## Ordre clarify minimum IA (priorité sections)

1. **Vision** — une phrase problème + solution  
2. **Target users** — qui paie / qui utilise  
3. **Core features** — 3 features MVP max  
4. *(IA peut skip le reste avant gen si suffisant)*  

Post-génération : vérifier **business_model**, **timeline**, **risks** non vides.

---

## 12 sections — ce qu’investisseur lit en 3 min

| Section | Pitch demain — contenu attendu |
|---------|-------------------------------|
| `executive_summary` | 5 lignes max |
| `vision` | Problème + insight |
| `target_users` | 1 persona |
| `core_features` | 3 bullets |
| `user_journeys` | 1 happy path |
| `technical` | Stack en 3 bullets |
| `success_metrics` | 2 métriques |
| `business_model` | Comment tu gagnes $ |
| `differentiation` | vs 1 concurrent |
| `timeline` | 90 jours |
| `out_of_scope` | 3 non-goals |
| `risks` | 2 risques honnêtes |

## Copy à dire en meeting

> « C’est une **version express** — à approfondir. La structure est là pour aligner la discussion demain. »

- Ne pas masquer le disclaimer share.
