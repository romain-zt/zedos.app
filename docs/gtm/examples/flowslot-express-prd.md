# FlowSlot — Express PRD (exemple galerie)

> Fictif — démo galerie Zedos. **Express — to be deepened.**

## executive_summary

FlowSlot is a B2B SaaS that helps independent gyms sell off-peak class spots via shareable same-day booking links. Phase 1: Île-de-France, 50 gyms, founder-led sales.

## vision

Gyms lose revenue on empty classes after 6pm. FlowSlot turns empty seats into cash without discount apps that train customers to wait.

## target_users

**Primary:** Gym owner, 1–3 locations, 200–800 members, no dedicated ops manager.

## core_features

1. Same-day class catalog (owner sets caps)  
2. One-link checkout (no app install for attendee)  
3. End-of-day payout summary  

## user_journeys

Owner publishes tonight’s 3 classes → shares link in Instagram story → attendee pays → gym sees booking in dashboard.

## technical

Next.js, Stripe Connect, Postgres, SMS optional v2.

## success_metrics

- 30% fill rate on listed off-peak classes within 60 days  
- &lt; 2 min owner time to publish daily slots  

## business_model

€49/mo per location + 5% on transactional volume.

## differentiation

vs Mindbody: no annual contract, same-day only, no consumer app.

## timeline

- M1: MVP + 5 pilots  
- M2: 20 gyms  
- M3: Self-serve onboarding  

## out_of_scope

Consumer loyalty program, equipment booking, nutrition coaching.

## risks

Gym staff won't publish daily — mitigate with WhatsApp reminder bot (v2).
