# Programme parrainage — 1 mois Pro offert

## Objectif

Accélérer viralité **après** premiers shares réussis — pas avant PMF express.

---

## Mécanique

| Acteur | Action | Reward |
|--------|--------|--------|
| **Parrain** (Builder ou Pro actif) | Partage lien `zedos.app/r/{code}` | **1 mois Pro** crédité après activation filleul |
| **Filleul** | Signup + 1er PRD généré (express ou standard) | **-20 %** premier mois Builder *(stackable avec founding? Non)* |

---

## Éligibilité parrain

- Compte **payant** ≥ 14 jours
- A créé ≥ 1 share link
- Max **10** filleuls récompensés / trimestre (anti-abus)

---

## Éligibilité filleul

- Nouveau compte (email + device fingerprint si dispo)
- Pas déjà en concierge founding cohort

---

## Anti-abus

| Règle | Détail |
|-------|--------|
| Auto-parrainage | Interdit (même email domaine jetable list) |
| Filleul fantôme | PRD généré minimum (pas signup seul) |
| Revocation | Zedos peut retirer crédits si fraude |

---

## Ops manuel (avant automatisation)

1. Parrain demande code via Typeform.  
2. Ops génère code Stripe coupon ou crédit manuel.  
3. Sheet : parrain / filleul / date / statut.
